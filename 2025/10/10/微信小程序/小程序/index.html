为什么选择 Uniapp + Vue3？

  • Uniapp：解决“多端适配”痛点，一套代码可发布微信小程序、支付宝小程序、H5、APP，不用为不同端重复开发，对小程序开发友好度拉满，完美兼容 Vue3 语法。
  • Vue3：相比 Vue2，Composition API 更灵活，逻辑复用更高效，配合 Vite 热更新，开发体验远超原生小程序的语法。
  • VS Code：轻量、插件丰富，熟悉的编码环境不用切换，搭配 uniapp 插件，可实现代码提示、语法高亮、快速跳转，比 HBuilderX 更贴合前端开发者的使用习惯。
  • 微信开发者工具：官方唯一指定调试、预览、上传工具，负责小程序的模拟器渲染、真机调试和发布，不可或缺。
  • Uniapp 官方文档：https://uniapp.dcloud.io/（重点看微信小程序适配部分）
  • Vue3 官方文档：https://cn.vuejs.org/（重点学习Composition API）
  • 微信小程序官方文档：https://developers.weixin.qq.com/miniprogram/dev/framework/（了解小程序原生规范）

优势

  • 真正跨端：一套代码跑小程序、App、H5、快应用
  • Vue3 语法：Composition API 更优雅、逻辑复用更强
  • 生态完善：插件多、坑少、社区活跃 -
  • 开发效率高：比原生小程序写法更简洁、更现代化 -
  • 热更新快：Vite 驱动，保存即更新，开发体验极佳

核心优势

  • 低门槛接入：无需应用商店审核，通过微信公众平台即可快速发布
  • 跨平台兼容：一套代码可运行在 iOS、Android 及微信 PC 端
  • 原生体验：接近原生 App 的流畅度与交互体验
  • 流量红利：可通过微信聊天、朋友圈、搜一搜等多场景触达用户
  • 开发高效：基于微信自定义的语法与框架，学习成本低、开发周期短

整体思路

1.通过 uni-app 提供的使用命令行构建项目的方式，构建出 Vue3 + TS 的 uni-app 项目（uni-app 自身就是基于 Vue.js 的跨平台开发框架）
2.Vscode 中安装 uni-app 插件，弥补相比 HBuilderX 缺少的便捷操作 3.安装相关的 @types 依赖，更好的支持 typescript 4.通过提供的编译脚本，将项目编译成期望的最终格式：将 Vue3 + TS 开发的 uni-app 项目编译成微信小程序项目（wxml、wxss）
微信开发者工具导入编译后的小程序项目，展示最终效果。

原理：

移动端设备的安全区域

现代移动端设备，尤其是一些全面屏手机，存在屏幕异形区域，比如刘海屏、水滴屏以及底部的虚拟按键区域。为了避免应用内容被这些异形区域遮挡，操作系统（如 iOS、Android 等）提供了安全区域的概念。安全区域指的是屏幕中可以正常显示内容，不会被状态栏、刘海、虚拟按键等遮挡的区域。

uni-app 对安全区域的默认处理

uni-app 是一个跨端开发框架，为了兼容不同端的安全区域，在 App 端（app-plus）有默认的安全区域处理策略。在默认情况下，uni-app 会自动对页面内容进行一定的安全区域适配，比如给页面底部预留一定的空间，以防止内容被底部虚拟按键遮挡。

safearea 配置项的作用

在 app-plus 配置中添加 “safearea”: {“bottom”: {“offset”: “none”}} ，其含义和作用如下：
safearea 字段：用于配置安全区域相关的设置。
bottom 子字段：专门针对屏幕底部的安全区域进行配置。
offset: “none”：明确告诉 uni-app，对于屏幕底部的安全区域，不需要额外添加偏移量。也就是说，不采用框架默认的在底部添加安全区域偏移的策略。

解决 uni-popup 底部空隙的原理

当使用 uni-popup 组件时，如果 uni-app 按照默认的安全区域适配策略，在底部添加了额外的偏移量，而 uni-popup 组件自身的样式和布局并没有考虑到这部分额外的偏移，就可能导致底部出现空隙。

简单来说，添加 “safearea”: {“bottom”: {“offset”: “none”}} 配置后，取消了框架在底部安全区域的默认偏移量，使得 uni-popup 组件可以按照原本的样式和布局正常显示，从而消除了底部的空隙。

VS Code 无 Uniapp 代码提示，语法报错

  1. 卸载冲突插件：关闭 VS Code，卸载 Vetur（Vue2 插件），避免与 Vue3 插件冲突，优先使用 Volar 插件。
  2. 安装必备插件：打开 VS Code 插件市场，搜索并安装「UniApp Vue3 Snippets」（uniapp 代码提示）、「Vue Language Features (Volar)」（Vue3 语法支持），安装完成后重启 VS Code。

  • getSnapshotBeforeUpdate() еңЁжңҖиҝ‘дёҖж¬ЎжёІжҹ“иҫ“еҮәпјҲжҸҗдәӨеҲ° DOM иҠӮзӮ№пјүд№ӢеүҚи°ғз”ЁгҖӮе®ғдҪҝеҫ—з»„д»¶иғҪеңЁеҸ‘з”ҹжӣҙж”№д№ӢеүҚд»Һ DOM дёӯжҚ•иҺ·дёҖдәӣдҝЎжҒҜпјҲдҫӢеҰӮпјҢж»ҡеҠЁдҪҚзҪ®пјүгҖӮжӯӨз”ҹе‘Ҫе‘Ёжңҹзҡ„д»»дҪ•иҝ”еӣһеҖје°ҶдҪңдёәеҸӮж•°дј йҖ’з»ҷ componentDidUpdate()
  • жӯӨз”Ёжі•е№¶дёҚеёёи§ҒпјҢдҪҶе®ғеҸҜиғҪеҮәзҺ°еңЁ UI еӨ„зҗҶдёӯпјҢеҰӮйңҖиҰҒд»Ҙзү№ж®Ҡж–№ејҸеӨ„зҗҶж»ҡеҠЁдҪҚзҪ®зҡ„иҒҠеӨ©зәҝзЁӢзӯү

  • жҖқиҖғпјҡеҰӮжһңдёӨдёӘз»„д»¶дёӯзҡ„йғЁеҲҶеҠҹиғҪзӣёдјјжҲ–зӣёеҗҢпјҢиҜҘеҰӮдҪ•еӨ„зҗҶпјҹ
  • еӨ„зҗҶж–№ејҸпјҡеӨҚз”Ёзӣёдјјзҡ„еҠҹиғҪ
  • еӨҚз”Ёд»Җд№Ҳпјҹ

  • еҰӮдҪ•жӢҝеҲ°иҜҘз»„д»¶дёӯеӨҚз”Ёзҡ„state

    • еңЁдҪҝз”Ёз»„д»¶ж—¶пјҢж·»еҠ дёҖдёӘеҖјдёәеҮҪж•°зҡ„propпјҢйҖҡиҝҮеҮҪж•°еҸӮж•°жқҘиҺ·еҸ–

  • еҰӮдҪ•жёІжҹ“еҲ°д»»ж„Ҹзҡ„UI

    • дҪҝз”ЁиҜҘеҮҪж•°зҡ„иҝ”еӣһеҖјдҪңдёәиҰҒжёІжҹ“зҡ„UIеҶ…е®№

  • е°ҶиҰҒеӨҚз”Ёзҡ„зҠ¶жҖҒдҪңдёә props.render(state)ж–№жі•зҡ„еҸӮж•°пјҢжҡҙйңІеҲ°з»„д»¶еӨ–йғЁ
  • дҪҝз”Ёprops.render() зҡ„иҝ”еӣһеҖјдҪңдёәиҰҒжёІжҹ“зҡ„еҶ…е®№

зӨәдҫӢdemo

childrenд»ЈжӣҝrenderеұһжҖ§

  • жҠҠpropжҳҜдёҖдёӘеҮҪж•°е№¶дё”е‘ҠиҜүз»„д»¶иҰҒжёІжҹ“д»Җд№ҲеҶ…е®№зҡ„жҠҖжңҜеҸ«еҒҡпјҡ render propsжЁЎејҸ

й«ҳйҳ¶з»„д»¶ пјҲвҳ…вҳ…вҳ…пјү

зӣ®ж Ү

  • зҹҘйҒ“й«ҳйҳ¶з»„д»¶зҡ„дҪңз”Ё
  • иғҪеӨҹиҜҙеҮәй«ҳйҳ¶зҡ„дҪҝз”ЁжӯҘйӘӨ

жҰӮиҝ°

  • зӣ®зҡ„пјҡе®һзҺ°зҠ¶жҖҒйҖ»иҫ‘еӨҚз”Ё
  • йҮҮз”Ё еҢ…иЈ…жЁЎејҸ
  • жүӢжңәпјҡиҺ·еҸ–дҝқжҠӨеҠҹиғҪ
  • жүӢжңәеЈіпјҡжҸҗдҫӣдҝқжҠӨеҠҹиғҪ
  • й«ҳйҳ¶з»„д»¶е°ұзӣёеҪ“дәҺжүӢжңәеЈіпјҢйҖҡиҝҮеҢ…иЈ…з»„д»¶пјҢеўһејәз»„д»¶еҠҹиғҪ

жҖқи·ҜеҲҶжһҗ

  • й«ҳйҳ¶з»„д»¶(HOCгҖҒHigher-Order Component) жҳҜдёҖдёӘеҮҪж•°пјҢжҺҘж”¶иҰҒеҢ…иЈ…зҡ„з»„д»¶пјҢиҝ”еӣһеўһејәеҗҺзҡ„з»„д»¶

  • й«ҳйҳ¶з»„д»¶еҶ…йғЁеҲӣе»әдәҶдёҖдёӘзұ»з»„д»¶пјҢеңЁиҝҷдёӘзұ»з»„д»¶дёӯжҸҗдҫӣеӨҚз”Ёзҡ„зҠ¶жҖҒйҖ»иҫ‘д»Јз ҒпјҢйҖҡиҝҮpropе°ҶеӨҚз”Ёзҡ„зҠ¶жҖҒдј йҖ’з»ҷиў«еҢ…иЈ…з»„д»¶WrappedComponent

дҪҝз”ЁжӯҘйӘӨ

  • еҲӣе»әдёҖдёӘеҮҪж•°пјҢеҗҚз§°зәҰе®ҡд»ҘwithејҖеӨҙ
  • жҢҮе®ҡеҮҪж•°еҸӮж•°пјҢеҸӮж•°еә”иҜҘд»ҘеӨ§еҶҷеӯ—жҜҚејҖеӨҙ
  • еңЁеҮҪж•°еҶ…йғЁеҲӣе»әдёҖдёӘзұ»з»„д»¶пјҢжҸҗдҫӣеӨҚз”Ёзҡ„зҠ¶жҖҒйҖ»иҫ‘д»Јз ҒпјҢе№¶иҝ”еӣһ
  • еңЁиҜҘз»„д»¶дёӯпјҢжёІжҹ“еҸӮж•°з»„д»¶пјҢеҗҢж—¶е°ҶзҠ¶жҖҒйҖҡиҝҮpropдј йҖ’з»ҷеҸӮж•°з»„д»¶
  • и°ғз”ЁиҜҘй«ҳйҳ¶з»„д»¶пјҢдј е…ҘиҰҒеўһејәзҡ„з»„д»¶пјҢйҖҡиҝҮиҝ”еӣһеҖјжӢҝеҲ°еўһејәеҗҺзҡ„з»„д»¶пјҢе№¶е°Ҷе…¶жёІжҹ“еҲ°йЎөйқў

еҢ…иЈ…еҮҪж•°

е“ӘдёӘз»„д»¶йңҖиҰҒеҠ ејәпјҢйҖҡиҝҮи°ғз”ЁwithMouseиҝҷдёӘеҮҪж•°пјҢз„¶еҗҺжҠҠиҝ”еӣһзҡ„еҖји®ҫзҪ®еҲ°зҲ¶з»„д»¶дёӯеҚіеҸҜ

и®ҫзҪ®displayName

  • дҪҝз”Ёй«ҳйҳ¶з»„д»¶еӯҳеңЁзҡ„й—®йўҳпјҡеҫ—еҲ°дёӨдёӘз»„д»¶зҡ„еҗҚз§°зӣёеҗҢ
  • еҺҹеӣ пјҡй»ҳи®Өжғ…еҶөдёӢпјҢReactдҪҝз”Ёз»„д»¶еҗҚз§°дҪңдёәdisplayName
  • и§ЈеҶіж–№ејҸпјҡдёәй«ҳйҳ¶з»„д»¶и®ҫзҪ®displayNameпјҢдҫҝдәҺи°ғиҜ•ж—¶еҢәеҲҶдёҚеҗҢзҡ„з»„д»¶
  • displayNameзҡ„дҪңз”Ёпјҡз”ЁдәҺи®ҫзҪ®и°ғиҜ•дҝЎжҒҜ(React Developer ToolsдҝЎжҒҜ)
  • и®ҫзҪ®ж–№ејҸпјҡ

дј йҖ’props

  • й—®йўҳпјҡеҰӮжһңжІЎжңүдј йҖ’propsпјҢдјҡеҜјиҮҙpropsдёўеӨұй—®йўҳ
  • и§ЈеҶіж–№ејҸпјҡ жёІжҹ“WrappedComponentж—¶пјҢе°Ҷstateе’ҢpropsдёҖиө·дј йҖ’з»ҷз»„д»¶

е°Ҹз»“

  • з»„д»¶йҖҡи®ҜжҳҜжһ„е»әReactеә”з”Ёеҝ…дёҚеҸҜе°‘зҡ„дёҖзҺҜ
  • propsзҡ„зҒөжҙ»жҖ§и®©з»„д»¶жӣҙеҠ ејәеӨ§
  • зҠ¶жҖҒжҸҗеҚҮжҳҜReactз»„д»¶зҡ„еёёз”ЁжЁЎејҸ
  • з»„д»¶з”ҹе‘Ҫе‘ЁжңҹжңүеҠ©дәҺзҗҶи§Јз»„д»¶зҡ„иҝҗиЎҢиҝҮзЁӢ
  • й’©еӯҗеҮҪж•°и®©ејҖеҸ‘иҖ…еҸҜд»ҘеңЁзү№е®ҡзҡ„ж—¶жңәжү§иЎҢжҹҗдәӣеҠҹиғҪ
  • render props жЁЎејҸе’Ңй«ҳйҳ¶з»„д»¶йғҪеҸҜд»Ҙе®һзҺ°з»„д»¶зҠ¶жҖҒйҖ»иҫ‘зҡ„еӨҚз”Ё
  • з»„д»¶жһҒз®ҖжЁЎеһӢпјҡ (state,props) => UI

ReactеҺҹзҗҶ

зӣ®ж Ү

  • иғҪеӨҹзҹҘйҒ“setState()жӣҙж–°ж•°жҚ®жҳҜејӮжӯҘзҡ„
  • иғҪеӨҹзҹҘйҒ“JSXиҜӯжі•зҡ„иҪ¬еҢ–иҝҮзЁӢ

setState()иҜҙжҳҺ пјҲвҳ…вҳ…вҳ…пјү

жӣҙж–°ж•°жҚ®

  • setState()жӣҙж–°ж•°жҚ®жҳҜејӮжӯҘзҡ„
  • жіЁж„ҸпјҡдҪҝз”ЁиҜҘиҜӯжі•пјҢеҗҺйқўзҡ„setStateдёҚиҰҒдҫқиө–еүҚйқўsetStateзҡ„еҖј
  • еӨҡж¬Ўи°ғз”ЁsetStateпјҢеҸӘдјҡи§ҰеҸ‘дёҖж¬Ўrender

жҺЁиҚҗиҜӯжі•

  • жҺЁиҚҗпјҡдҪҝз”Ё setState((state,props) => {}) иҜӯжі•
  • еҸӮж•°stateпјҡ иЎЁзӨәжңҖж–°зҡ„state
  • еҸӮж•°propsпјҡ иЎЁзӨәжңҖж–°зҡ„props

з¬¬дәҢдёӘеҸӮж•°

  • еңәжҷҜпјҡеңЁзҠ¶жҖҒжӣҙж–°(йЎөйқўе®ҢжҲҗйҮҚж–°жёІжҹ“)еҗҺз«ӢеҚіжү§иЎҢжҹҗдёӘж“ҚдҪң
